About

This is a statistics website for my Auto Host Rotate multiplayer lobby. Data is collected automatically by an Auto Host bot in real time. The site and the bot are under active development.

Commands

!skipVote to skip the current host (55% of players). The host can skip themselves instantly.
!abortVote to abort the match (55% of playing players). Only works during a match.
!startVote to start the match (55% of players). Match also starts automatically when all players are ready.
!queue / !qShow the host queue.
!statsYour stats: matches played, wins, lobby rank.
!rankYour win count and rank by wins.
!playtimeYour time in lobby and rank by playtime.
!topTop 5 players by wins.
!website / !wLink to this website.
!flipVote to shuffle the queue (80% of players).
!bugreport <text>Send a bug report to the host.
!helpList of all commands.

Some commands may not be implemented yet.

Rules & Info
